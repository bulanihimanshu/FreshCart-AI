# 07-02: Metrics Computation & Export

<objective>
Compute HR@5, HR@10, Precision@5, MRR, and Catalog Coverage % uniformly for all 4 systems (Apriori, LSTM, LSTM+Time, Full System) across the 1000 test users, and export the comparison table to `results.csv`.
</objective>

<tasks>
  <task>
    <description>Define evaluation metric functions</description>
    <read_first>
      - .planning/phases/07-full-evaluation/07-CONTEXT.md
    </read_first>
    <action>
      Implement scoring functions that compute:
      - `HR@5` and `HR@10` (Hit Rate)
      - `Precision@5`
      - `MRR` (Mean Reciprocal Rank)
      - `Coverage` (total unique product IDs recommended across all users divided by the 5000 valid vocab size)
    </action>
    <acceptance_criteria>
      - Function `evaluate_coverage(predictions, subset_size=5000)` defined.
      - Functions `hit_rate(y_true, y_pred, k)` and `mrr(y_true, y_pred)` defined.
    </acceptance_criteria>
  </task>

  <task>
    <description>Run batch inference and compute scores for all systems</description>
    <action>
      Create a master evaluation loop that:
      1. Loads Apriori metrics from `baseline_results.csv`. (Since Apriori was evaluated offline, copy its metrics. For Apriori coverage, default to 20/5000 since it always recommends the same top globals failing personalized rules).
      2. Runs `lstm_model` batch inference on unmodified `X_test`, evaluates all metrics.
      3. Runs `lstm_time_model` batch inference on unmodified `X_test` with synthetic time features, evaluates all metrics.
      4. Runs `get_recommendations` (Full System router) systematically over `X_test_simulated`, evaluates all metrics.
    </action>
    <acceptance_criteria>
      - 4 discrete sets of metrics are compiled into a list of dictionaries.
    </acceptance_criteria>
  </task>

  <task>
    <description>Format and export results.csv</description>
    <action>
      Create a Pandas DataFrame from the compiled metrics.
      Set the columns to: `['System', 'HR@5', 'HR@10', 'Precision@5', 'MRR', 'Coverage (%)']`.
      Round all floats to 4 decimal places.
      Export DataFrame to `data/processed/results.csv`, saving index=False.
    </action>
    <acceptance_criteria>
      - `data/processed/results.csv` contains exactly 4 lines (plus header).
      - Header matches exactly the required columns.
    </acceptance_criteria>
  </task>
</tasks>

<verification>
