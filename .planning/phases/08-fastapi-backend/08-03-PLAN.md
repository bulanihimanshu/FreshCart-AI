# Product Search and Details

## Context
Endpoints for querying products. To safeguard the ML model, the search endpoint MUST filter out any products not in the vocabulary (D-01).

## Verification criteria
- [ ] `GET /api/products?query=milk` returns items from the vocabulary limits to `limit`
- [ ] must_haves:
  - Product results must be successfully hydrated from app.state pre-loaded products database.
  - Search prevents OOV (out of vocabulary) items.
